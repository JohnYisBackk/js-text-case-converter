# Text Case Converter PRO

A clean and modern **Text Case Converter** built with **HTML, CSS and JavaScript**.

This tool allows users to transform text instantly and perform common text editing utilities in a simple and elegant interface.

---

## ✨ Features

- Convert text to **UPPERCASE**
- Convert text to **lowercase**
- **Capitalize** the first letter of each word
- **Remove extra spaces**
- **Copy converted text** to clipboard
- **Clear all text**
- Live **character counter**
- Live **word counter**
- Clean and responsive UI

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 How It Works

1. Enter text in the input area.
2. Choose one of the text transformation options.
3. The converted text appears instantly in the output area.
4. You can copy the result or clear the editor.
